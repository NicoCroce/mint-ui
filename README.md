## Install

```bash
yarn add @turtlemint/mint-ui
# or npm install @turtlemint/mint-ui --save
```


## Basic Usage

```jsx
import { Button } from '@turtlemint/turtlemint-ui';
ReactDOM.render(<Button />, mountNode);
```
